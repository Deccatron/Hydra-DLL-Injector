To use Hydra follow these steps

1) When prompted to "Enter the Process ID:" simply use a tool like Task manager or Process Hacker 2 to find the PID of the program then simply paste or type it into the prompt then follow by hitting enter.

2) When prompted to "Enter the path to the DLL:" simply drag and drop your dll you would like to inject into the process into the Hydra window (MAKE SURE TO THEN REMOVE THE QUOTATION MARKS AROUND THE FILE PATH) then follow by hitting enter.

Now your done your DLL file should have succesfully injected into your target process!
